1. Вступ

«Readmore.js» - це плагін JavaScript, який дозволяє створювати функціональність «Читати далі» для вашого веб-контенту. Він розроблений для зроблення довгих блоків тексту або контенту розширюваними за допомогою кнопки «Читати далі» для кращого взаємодії з користувачами.

Особливості

  • Автоматично додає кнопку «Читати далі» до довгих блоків контенту;
  • Плавне розширення та згортання контенту з настроюваною анімацією;
  • Налаштовуваний розмір згорнутого контенту, швидкість анімації та інше;
  • Підтримує адаптивний дизайн для активації функціональності в залежності від ширини екрану.

2. Встановлення

Щоб використовувати «Readmore.js», вам потрібно включити скрипт у свій HTML-файл:


						<script src="readmore.js"></script>

Переконайтеся, що цей тег скрипту розміщено в розділі <head> вашого HTML-документа.

3. Основне використання

3.1 Структура HTML

Створіть HTML-елемент, який ви хочете зробити розширюваним. Дайте йому конкретний клас, наприклад, «readmore»:


						<div class="readmore">
							<!-- Ваш контент тут -->
						</div>

3.2 Ініціалізація JavaScript

Ініціалізуйте плагін «Readmore.js» у вашому JavaScript-коді. Наведений нижче код показує, як це зробити:


					document.addEventListener('DOMContentLoaded', function () {
						initReadMore('.readmore', {
							collapsedHeight: 200,  // Налаштування розміру згорнутого контенту (в пікселях)
							speed: 100,            // Налаштування швидкості анімації (в мілісекундах)
							heightMargin: 16,      // Налаштування відступу при розширенні (в пікселях)
							moreLink: '<a href="#">Читати далі</a>',  // Налаштування HTML для «Читати далі»
							lessLink: '<a href="#">Закрити</a>'      // Налаштування HTML для «Закрити»
						}, 768);  // Необов'язково: Встановлення максимальної ширини для адаптивної поведінки
					});

3.3 Функціональність

При перегляді вашої сторінки, якщо вміст всередині цього елемента вищий за вказаний collapsedHeight, буде автоматично додана кнопка «Читати далі». Клацання цієї кнопки розгорне вміст, а подальше клацання знову згорне його.

4. Користувацька настройка

Ви можете налаштовувати поведінку та вигляд плагіну «Readmore.js», змінюючи опції, передані у функцію initReadMore.

  • collapsedHeight: Налаштування розміру згорнутого контенту (в пікселях);
  • speed: Встановлення швидкості анімації при розгортанні або згортанні (в мілісекундах);
  • heightMargin: Встановлення відступу при розширенні (в пікселях);
  • moreLink: Налаштування HTML для посилання «Читати далі»;
  • lessLink: Налаштування HTML для посилання «Закрити».

Не соромтесь налаштовувати ці параметри відповідно до вашого дизайну та потреб.

5. Адаптивна поведінка

Для активації адаптивної поведінки ви можете передати параметр maxWidth під час ініціалізації плагіну. Якщо ширина екрану менше вказаного maxWidth, функціональність «Читати далі» буде активною.


					initReadMore('.readmore', options, 768);

Це дозволить вам контролювати, коли функціональність «Читати далі» активна на основі ширини екрану.

6. Розширене використання

Для більш розширеного використання та інтеграції з іншими скриптами або бібліотеками ви можете безпосередньо викликати функцію initReadMore з вашими параметрами.


					initReadMore('.custom-class', {
					   // Користувацькі параметри
					});
