ReadMore.js
Просте та ефективне рішення на чистому JS

Плавний і чутливий плагін на чистому JavaScript для згортання та розгортання довгих текстових блоків. Кнопки «Читати далі» та «Приховати» дозволяють приховувати або показувати додатковий контент, покращуючи зручність читання і заощаджуючи місце на сторінці. Ідеально підходить для мобільних пристроїв та зручного управління великими обсягами тексту.

npm i @corgras/readmore-js
