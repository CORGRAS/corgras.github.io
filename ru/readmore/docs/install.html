Начало работы

Есть несколько способов интегрировать Readmore.js в ваш проект. Выберите наиболее подходящий для ваших потребностей.

Подключение через NPM

Это удобный вариант для больших проектов, где используется NPM. Использование NPM также позволяет легко обновлять библиотеку и управлять зависимостями вашего проекта, а также интегрировать ее в большие системы на базе JavaScript, такие как React или Vue.js.

— В командной строке выполните:


					$ npm i @corgras/readmore-js

— После установки вы можете импортировать функцию initReadMore в вашем JavaScript файле.


					import { initReadMore } from '@corgras/readmore-js';
					// Остальной код

— Затем вы можете вызвать initReadMore с необходимыми параметрами.

Ручное подключение

Этот способ позволяет добавить файл непосредственно в ваш HTML-код. Просто загрузите скрипт, сохраните его в папке вашего проекта и подключите к документу. Такое подключение дает вам полный контроль над версией скрипта, которую вы используете.

— Загрузите файлы скрипта Readmore.js с GitHub

— После загрузки файла в определенную папку на сайте, нужно подключить его к HTML-странице. Вставьте приведенный ниже код в секцию <head> или перед закрытием тега </body> на вашей HTML-странице:


					<script src="путь_к_вашей_папке/readmore.min.js" defer></script>

В теге <script> используется атрибут defer. Скрипт будет выполнен после загрузки HTML, но перед событием DOMContentLoaded.

— После подключения вам нужно инициализировать скрипт, чтобы он начал работать с выбранными элементами на вашей странице.

Подключение через CDN

Скрипт можно подключить с удаленного сервера, что значительно ускорит загрузку вашего сайта и устраняет необходимость хранить его локально. Использование CDN обеспечивает доступ к кэшированным версиям скрипта из разных уголков мира. Вы можете воспользоваться одним из популярных сервисов CDN, таких как jsDelivr или Unpkg.

— Для подключения вставьте один из приведенных ниже строк кода в секцию <head> или перед закрытием тега </body> на вашей HTML-странице:

CDN jsDelivr:


					<script src="https://cdn.jsdelivr.net/npm/@corgras/readmore-js@1.0.0/readmore.min.js" defer></script>

CDN Unpkg:


					<script src="https://unpkg.com/@corgras/readmore-js@1.0.0/readmore.min.js" defer></script>

В теге <script> используется атрибут defer. Скрипт будет выполнен после загрузки HTML, но перед событием DOMContentLoaded.

— После подключения вам нужно инициализировать скрипт, чтобы он начал работать с выбранными элементами на вашей странице.
