Текстовый блок HTML PLUS

Позволяет отображать произвольный HTML-код на страницах сайта с возможностью скрывать / развертывать текст и управлять отображением текста для определенных групп покупателей

Возможности модуля

  • Локализация 3 языков: Украинский, Английский, Русский;
  • Управление скриптом «Подробнее»;
  • Управление уровнем заголовка H1-H6;
  • Использование кастомного шаблона для модуля;
  • Управление доступом к контенту;
  • Мультиязычность;
  • Вставка изображений для разных размеров экранов;
  • Управление размерами изображения Новое;
  • Отложенная загрузка изображений средствами браузера;
  • Вставка видео;
  • Размещение контента по изображению или видео;
  • Управление на какие разрешения экранов будет работать скрипт.

Установка модуля

Модификатор не устанавливается?

Установите «Localcopy OCMOD», после чего попробуйте установить модификатор снова.

Загрузить
  • В Админ-панели перейдите «Меню -> Расширения -> Установка расширений» и установите файл «HTML_Content_Plus_x_OpenCart_x.ocmod.zip»;
  • Получаем сообщение об успешной установке, переходим «Меню -> Система -> Группы пользователей» и для администратора установить права доступа (в блоках «Разрешить просмотр» и «Разрешить редактировать» отметить галочкой «extension/module/cs_html_content_plus» );
  • В списке модулей «Меню -> Расширение -> Модули» установите модуль «Текстовий блок HTML PLUS».

Настройка модуля

Для настройки модуля перейдите «Меню -> Расширения -> Модули -> Текстовый блок HTML PLUS»

Общие:

  • Название модуля - название, которое будет отображаться в админ-панели;
  • Статус - отвечает за включение модуля;
  • Изображение - вставка изображений для разных размеров экранов или одно для всех;
  • Видео список - выбор откуда будет вставлен видео;
  • Ссылка на видео - для поддерживаемых сервисов достаточно вставить любую ссылку на видео для других скопировать ссылки с « iframe src="" ».

Настройки:

  • Позиция контента - расположение текста в отношении изображения или видео;
  • Уровень заголовка - уровень заголовка для отображения на сайте;
  • Статус кнопки «Подробнее» - включение / выключение скрипта;
  • Высота видимого текста - которая будет читаема, а остальная часть текста будет скрыта;
  • Расширение экрана - на каких разрешениях экрана будет работать скрипт;
  • Изменить размер большого изображения - включение изменения размера под собственные / выключение для отображения исходного размера;
  • Размер большого изображения (Ш x В) - для больших экранов;
  • Изменить размер среднего изображения - включение изменения размера под собственные + если изображение для средних экранов не добавлено, то будет создано из большого. Выключение для отображения исходного размера;
  • Размер среднего изображения (Ш x В) - для средних экранов;
  • Изменить размер маленького изображения - включение изменения размера под собственные + если изображение для маленьких экранов не добавлено, то будет создано из большого. Выключение для отображения исходного размера;
  • Размер маленького изображения (Ш x В) - для маленьких экранов.

Доступ:

  • Доступ - кому будет виден этот блок: для всех пользователей / только для зарегистрированных / только для гостей;
  • Группы покупателей - управление доступом к группам клиентов.

Дизайн:

  • Шаблон - после создания шаблона для модуля, вы можете выбрать его здесь.

Шаблон

Создайте шаблон модуля и разместите его в папке:

« catalog / view / theme / default / template / custom_templates / module / html_content_plus / имя файла на латинице .(twig.tpl) »

За основу можете взять стандартный шаблон по пути:

« catalog / view / theme / default / template / extension / module / cs_html_content_plus.(twig.tpl) »

Стилизация

Если стандартный дизайн кнопки «Развернуть / Свернуть» не вписывается в ваш дизайн сайта, вы можете подправить CSS стили в файле «cs_html_content_plus.min.css», который находится:

« catalog / view / javascript / cs_libs / css / cs_html_content_plus.min.css »
