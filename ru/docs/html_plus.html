Текстовый блок HTML PLUS

Позволяет отображать произвольный HTML-код на страницах сайта с возможностью скрывать/развертывать текст и управлять отображением текста для определенных групп покупателей.

Версия: 4.0
OpenCart: 2.1 | 2.2 | 2.3 | 3.x
Релиз: 29.11.2022

Возможности модуля

  • Поддержка 3 языков: Украинский, Английский, Русский;
  • Управление скриптом «Подробнее»;
  • Управление на какие разрешения экранов будет работать скрипт;
  • Управление уровнем заголовка H1-H6;
  • Использование кастомного шаблона для модуля;
  • Управление доступом к контенту;
  • Мультиязычность.

Установка расширения

  • В Админ-панели перейдите «Расширения -> Установка расширений» и выберите файл «HTML_Content_Plus_v4_OpenCart_3.ocmod.zip»;
  • Получаем сообщение об успешной установке, переходим «Система -> Группы пользователей» и для администратора установить права доступа (в блоках «Разрешить просмотр» и «Разрешить редактировать» отметить галочкой «extension/module/cs_html_plus»);
  • В списке модулей «Расширение -> Модули» установите модуль «Текстовий блок HTML PLUS».

Настройка расширения

Для настройки модуля перейдите: «Расширения -> Модули -> Текстовый блок HTML PLUS»

  • Название модуля - название, которое будет отображаться в админ-панели;
  • Статус - статус модуля отвечает за включение настроек;
  • Статус кнопки «Подробнее» - включение/выключение скрипта;
  • Высота видимого текста - которая будет читаема, а остальная часть текста будет скрыта;
  • Расширение экрана - на каких разрешениях экрана будет работать скрипт;
  • Уровень заголовка - уровень заголовка для отображения на сайте;
  • Доступ - кому будет виден этот блок: для всех пользователей / только для зарегистрированных / только для гостей;
  • Группы покупателей - управление доступом к группам клиентов;
  • Шаблон - после создания шаблона для модуля, вы можете выбрать его здесь.

Шаблон

Можете создать свой шаблон модуля, взяв за основу стандартный шаблон:

« catalog \ view \ theme \ default \ template \ extension \ module \ cs_html_plus.(twig.tpl) »

И разместить свой в папке:

« catalog \ view \ theme \ default \ template \ custom_templates \ module \ html_plus \ имя файла на латинице.(twig.tpl) »

Стилизация

Если стандартный дизайн кнопки «Развернуть/Свернуть» не вписывается в ваш дизайн сайта, вы можете подправить CSS стили в файле «cs_html_plus.min.css», который находится:

« catalog \ view \ cs_libs \ css \ cs_html_plus.min.css »
