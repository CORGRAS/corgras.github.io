Единицы измерения товара

Позволяет добавлять и настраивать различные единицы измерения цен для товаров и опций. Подойдет, если вы продаете товар, цена на который устанавливается за определенное его количество, то есть цена за кв.м., пог.м, м3, кг, шт. и т.д.

Версия: 1.0
OpenCart: 3.x
Релиз: 02.09.2023

Возможности расширения

  • Локализация 3 языков: Украинский, Английский, Русский;
  • Мультиязычность;
  • Возможность указать в карточке товара единицу измерения;
  • Возможность указать единицу измерения для каждой опции товара;
  • Возможность указать одну единицу измерения по-умолчанию для всех товаров;
  • Изменение единицы измерения при выборе опции;
  • Единицы выводятся только для информативности на странице товара, категорий, поиска, производителях, модулях.

Установка расширения

Важно!

При первой установке модуля, автоматически устанавливаются дополнительные поля в таблицы БД.
На всякий случай делайте бэкап БД.

Модификатор не устанавливается?

Установите «Localcopy OCMOD», после чего попробуйте установить модификатор снова.

Загрузить

Установка модификатора состоит из двух частей:

  • «Unit_Of_Measurement_Product_x_OpenCart_x.ocmod.zip» - содержится основной функционал модуля и модификатора;
  • «Unit_Of_Measurement_Product_Default_OpenCart_x.ocmod.zip» - архив для модификации шаблона.

Установка модификаторов:

  • В Админ-панели перейдите «Меню -> Расширения -> Установка расширений» и установите два архива;
  • Получаем сообщение об успешной установке, переходим «Меню -> Расширение -> Модификаторы» и в правом верхнем углу нажимаем кнопку «Обновить»;
  • После этого перейдите в «Меню -> Система -> Группы пользователей» и для администратора установить права доступа (в блоках «Разрешить просмотр» и «Разрешить редактировать» отметить галочкой «extension/module/cs_unit»);
  • В списке модулей «Меню -> Расширение -> Модули» установите модуль «Единицы измерения товара».

Добавление единицы измерения

Для того, чтобы добавить единицу измерения в OpenCart, необходимо выполнить следующие действия.

Описание действий:

  • Перейдите на страницу списка единиц измерения «Меню -> Система -> Локализация -> Единицы Измерения Товара»;
  • Нажмите кнопку «Добавить»;
  • В открывшемся окне необходимо заполнить форму;
  • Нажимаем кнопку «Сохранить»;

Настройка расширения

Для настройки расширения перейдите «Меню -> Расширения -> Модули -> Единицы Измерения Товара»

  • Статус - отвечает за включение расширения;
  • Разделитель - позволяет вывести символ, который будет между ценой и единицей измерения [Пример: 10.00 / шт. | 10.00 за м.кв ];
  • Ед. измерения для опций товара - позволяет назначать единицу измерения для каждой опции + вывод в шаблоне;
  • Ед. измерения в списке товаров (Админ-панель) - отображает колонку с единицей измерения в админ-панели на странице списка товаров + сортировка единицы измерения.

Стилизация

Если требуется стилизация элементов, то в CSS-файле который подключен к шаблону добавьте классы с нужными вам атрибутами:

  • .separate - стилизация разделителя;
  • .unit - стилизация единицы измерения;
  • .separate-option - стилизация разделителя в опциях;
  • .unit-option - стилизация единицы измерения в опциях.
