Информационные блоки

Позволяет отображать информационные страницы (О нас, Информация о доставке, Политика Безопасности, Условия соглашения и т.д.) в виде блоков с кратким описанием

Возможности модуля

  • Локализация 3 языков: Украинский, Английский, Русский;
  • Назначить изображение или иконку для блока;
  • Назначить количество колонок для разной ширины экрана;
  • Переопределить заголовок;
  • Переопределить краткое описание;
  • Управление количеством символов для краткого описания;
  • Управление размером изображения;
  • Управление доступом и группами покупателей;
  • Подключать шаблоны к модулю;
  • Мультиязычность;
  • Управлять ссылками (изображение / иконка, заголовок, кнопка).

Установка расширения

Модификатор не устанавливается?

Установите «Localcopy OCMOD», после чего попробуйте установить модификатор снова.

Загрузить
  • В Админ-панели перейдите «Меню -> Расширения -> Установка расширений» и установите файл «Information_Blocks_x_OpenCart_x.ocmod.zip»;
  • Получаем сообщение об успешной установке, переходим «Меню -> Система -> Группы пользователей» и для администратора установить права доступа (в блоках «Разрешить просмотр» и «Разрешить редактировать» отметить галочкой «extension/module/cs_info_blocks» );
  • В списке модулей «Меню -> Расширение -> Модули» установите модуль «Информационные блоки».

Настройка расширения

Для настройки модуля перейдите «Меню -> Расширения -> Модули -> Информационные блоки»

Общие настройки:

  • Название модуля - название, которое будет отображаться в админ-панели;
  • Статус - отвечает за включение модуля;

Настройка раздела:

  • Заголовок раздела - заголовок, который будет отображаться над блоками;
  • Колонки - которая будет читаема, а остальная часть текста будет скрыта;
  • Размеры изображения - ширина / высота изображения, или отключает изменение размера изображения - 0x0;
  • Знаков в описании - количество символов, отображаемых для описания;
  • Статус описания - статус отображения описания;
  • Ссылка в виде изображения - ссылка устанавливается на изображение или иконку;
  • Ссылка в виде заголовка - ссылка устанавливается на заголовок;
  • Ссылка в виде кнопки - ссылка устанавливается в виде кнопки.

Блоки:

  • Выберите статью - выбор из созданных в разделе «Категории -> Статьи»;
  • Изображение / иконка - выбор, показывать изображения, иконку или отключить;
  • Изображение - установка изображения;
  • Class иконки - классы иконок можно просмотреть на сайте FontAwesome;
  • Сортировка - порядок блоков;
  • Заголовок статьи - возможность переопределить заголовок статьи;
  • Описание - возможность переопределить описание статьи.

Доступ / Дизайн:

  • Доступ - кому будет виден этот блок: для всех пользователей / только для зарегистрированных / только для гостей;
  • Группы покупателей - управление доступом к группам клиентов;
  • Шаблон - после создания шаблона для модуля, вы можете выбрать его здесь.

Шаблон

Создайте шаблон модуля и разместите его в папке:

« catalog / view / theme / default / template / custom_templates / module / info_blocks / имя файла на латинице .(twig.tpl) »

За основу можете взять стандартный шаблон по пути:

« catalog / view / theme / default / template / extension / module / cs_info_blocks.(twig.tpl) »

Стилизация

Если стандартный дизайн не вписывается в ваш дизайн сайта, можно подправить стили CSS в файле «cs_info_blocks.min.css», который находится:

« catalog / view / cs_libs / css / cs_info_blocks.min.css »
