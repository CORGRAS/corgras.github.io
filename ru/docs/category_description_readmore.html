JS Readmore для описания категории

Позволяет эффективно управлять длинными описаниями категорий на вашем веб-сайте. С его помощью вы можете создавать компактные и информативные категорийные страницы, предоставляя пользователям возможность скрывать и разворачивать текст по своему усмотрению

Возможности расширения

  • Локализация 3 языков: Украинский, Английский, Русский;
  • Управление скриптом «Подробнее» для всех категорий или индивидуально;
  • Управление на каких разрешений экранов будет работать скрипт.

Для некоторых сайтов могут понадобиться некоторые доработки или адаптация под шаблон.

Установка расширения

Важно!

При первой установке модуля, автоматически устанавливаются дополнительные поля в таблицы БД.
На всякий случай делайте бэкап БД.

Модификатор не устанавливается?

Установите «Localcopy OCMOD», после чего попробуйте установить модификатор снова.

Загрузить
  • В Админ-панели перейдите «Меню -> Расширения -> Установка расширений» и установите файл «CD_Readmore_x_OpenCart_x.ocmod.zip»;
  • Получаем сообщение об успешной установке, переходим «Меню -> Расширение -> Модификаторы» и в правом верхнем углу нажимаем кнопку «Обновить»;
  • После этого перейдите в «Меню -> Система -> Группы пользователей» и для администратора установить права доступа (в блоках «Разрешить просмотр» и «Разрешить редактировать» отметить галочкой «extension/module/cs_cd_readmore» );
  • В списке модулей «Меню -> Расширение -> Модули» установите модуль «JS Readmore для описания категории».

Настройка расширения

Для настройки расширения, перейдите «Меню -> Расширения -> Модули -> JS Readmore для описания категории»

Настройки описания:

  • Статус- отвечает за включение расширения;
  • Статус кнопки «Подробнее» - включение / выключение скрипта для всех описаний категорий;
  • Высота видимого текста - которая будет читаемая, а другая часть текста будет скрыта;
  • Разрешение экрана - на каких разрешениях экрана будет работать скрипт.

Статус БД

  • Отображение статуса полей в таблице БД;
  • Управление устанавкой или удалением полей из БД.

Настройки в категориях

Чтобы настроить расширение для конкретной категории, перейдите «Меню -> Каталог -> Категории» и выберите нужную категорию. Дополнительные параметры находятся во вкладке «Дизайн»

  • Статус кнопки «Подробнее» - включить / выключить скрипт для конкретной категории или оставить по-умолчанию, тогда применяются настройки из модуля;
  • Высота видимого текста - которая будет читаемая, а другая часть текста будет скрыта.

Стилизация

Если стандартный дизайн кнопки «Развернуть / Свернуть» не вписывается в ваш дизайн сайта, вы можете подправить CSS стили в файле «cs_cd_readmore.min.css», который находится:

« catalog / view / javascript / cs_libs / css / cs_cd_readmore.min.css »
