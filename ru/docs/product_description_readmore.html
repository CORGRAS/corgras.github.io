JS Readmore для описания товара

Позволяет скрыть часть длинного текста, с возможностью скрывать / развертывать описание товара

Возможности модуля

  • Поддержка 3 языков: Украинский, Английский, Русский;
  • Управление скриптом «Подробнее» для всех товаров или индивидуально;
  • Управление на каких разрешений экранов будет работать скрипт, для всех или индивидуально.

Предупреждение

Модификатор тестировался на стандартном шаблоне и на чистой системе Opencart и ocStore.
Для некоторых сайтов могут понадобиться некоторые доработки или адаптация под шаблон.

Установка расширения

ВАЖНО!

При первой установке модуля, автоматически устанавливаются дополнительные поля в таблицы БД. На всякий случай делайте бэкап БД.

В Админ-панели перейдите «Расширения -> Установка расширений»:

  • Установите «PD_Readmore_v1_OpenCart_x.ocmod.zip»;
  • Получаем сообщение об успешной установке, переходим «Расширение -> Модификаторы» и в правом верхнем углу нажимаем кнопку «Обновить»;
  • После этого перейдите в «Система -> Группы пользователей» и для администратора установить права доступа (в блоках «Разрешить просмотр» и «Разрешить редактировать» отметить галочкой «extension/module/cs_pd_readmore»);
  • В списке модулей «Расширение -> Модули» установите модуль «JS Readmore для описания товара».

Настройка расширения

Для настройки модуля перейдите: «Расширения -> Модули -> JS Readmore для описания товара»

Настройки описания:

  • Статус описания - отображение блока с описанием на сайте;
  • Статус кнопки «Подробнее» - включение / выключение скрипта для всех описаний товаров;
  • Высота видимого текста - которая будет читаемая, а другая часть текста будет скрыта;
  • Разрешение экрана - на каких разрешениях экрана будет работать скрипт.

Статус БД

  • Отображение статуса полей в таблице БД;
  • Управление устанавкой или удалением полей из БД.

Настройка расширения

Дополнительные настройки находятся во вкладке «Дизайн»:

  • Статус кнопки «Подробнее» - включить / выключить скрипт для конкретного товара или оставить по-умолчанию, тогда применяются настройки из модуля;
  • Высота видимого текста - которая будет читаемая, а другая часть текста будет скрыта;

Стилизация

Если стандартный дизайн кнопки «Развернуть / Свернуть» не вписывается в ваш дизайн сайта, вы можете подправить CSS стили в файле «cs_pd_readmore.min.css», который находится:

« catalog \ view \ cs_libs \ css \ cs_pd_readmore.min.css »
