Module installation

The module does not install?

Install «Localcopy OCMOD», then try to install the module again.

Download
  • In the admin panel go to «Menu -> Extensions -> Installer» and install the file «HTML_Content_Pro_x_OpenCart_x.ocmod.zip»;
  • We get a message about successful installation, go to «Menu -> System -> User Groups» and set permissions for administrator (in blocks «Access Permission» and «Modify Permission» mark with a tick «extension/module/cs_html_content_pro» );
  • In the list of modules «Menu -> Extensions -> Extensions» install the module «HTML-Content PRO».

Module configuration

To configure the module, go to «Menu -> Extensions -> Extensions -> HTML-Content PRO»:

General:

  • Module name - name that will be displayed in the admin panel;
  • Status - responsible for enabling the module;

Section:

  • Image - insert images for different screen sizes or one for all;
  • Video list - choose where the video will be inserted;
  • Video link - for supported services, it is enough to insert any video link, for others, copy links from « iframe src="" ».

Items:

  • Image - insert images for different screen sizes or one for all;
  • Class icon - to display an icon, add its CLASS. See the documentation for the library that is connected;
  • Video list - choose where the video will be inserted;
  • Video link - for supported services, it is enough to insert any video link, for others, copy links from « iframe src="" »;
  • Positioning media elements - the position of the media elements relative to the text;
  • Category - link to the category;
  • Information - link to an informational article;
  • Manufacturer - link to manufacturer;
  • Other link - any other link;
  • Button name - the text to be displayed on the button;
  • Class - individual CSS class for the item;
  • Id - individual CSS id for the item;
  • Attributes - individual HTML attribute for the item.

Settings:

  • Content grid - in what order the blocks will be displayed;
  • Class - individual CSS class for the section;
  • Id - individual CSS id for the section;
  • Attributes - individual HTML attribute for the section;
  • Columns - setting the number of columns for different screens;
  • Text alignment in items - heading and text alignment in blocks.;
  • Resize large image - enable custom resizing / disable to display original size;
  • Size large image (W x H) - for larger screens;
  • Resize medium image - enable custom resizing + if an image for medium screens is not added, it will be created from a large one. Turn off to display the original size;
  • Size medium image (W x H) - for medium screens;
  • Resize small image - enable custom resizing + if an image for small screens is not added, it will be created from a large one. Turn off to display the original size;
  • Size small image (W x H) - for small screens.

Access:

  • Access - to whom this block will be visible: for all users / for registered users only / for guests only;
  • Customer groups - manage access to customer groups.

Design:

  • Template - After creating a template for a module, you can select it here.

Template

The module has four types of block display:

  • cs_html_content_pro - blocks are displayed as columns / each block can be a link (Template - by default);
  • cs_html_content_pro_item_with_button - blocks are displayed as columns / link as a button;
  • cs_html_content_pro_tabs - blocks are displayed as tabs;
  • cs_html_content_pro_accordion - blocks are displayed as an accordion.

It is also possible to create a new module template and place it in a folder:

« catalog / view / theme / default / template / custom_templates / module / html_content_pro / file name in latin .(twig.tpl) »

Stylization

The CSS styles are in the file «cs_html_content_pro.css» located:

« catalog / view / javascript / cs_libs / css / cs_html_content_pro.css »
