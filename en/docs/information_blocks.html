Information blocks

Allows you to display information pages (About Us, Shipping Information, Security Policy, Terms of Agreement, etc.) as blocks with a brief description

Module capabilities

  • Localization of 3 languages: Ukrainian, English, Russian;
  • Assign an image or icon to the block;
  • Assign number of columns for different screen widths;
  • Redefine title;
  • Override short description;
  • Manage the number of characters for a short description;
  • Image size control;
  • Manage links (image / icon, title, button);
  • Manage access and customer groups;
  • Include templates in a module;
  • Multilanguage.

Extension installer

The module does not install?

Install «Localcopy OCMOD», then try to install the module again.

  • In the admin panel go to «Extensions -> Installer» and select file «Information_Blocks_v3_OpenCart_3.ocmod.zip»;
  • We get a message about successful installation, go to «System -> User Groups» and set permissions for administrator (in blocks «Access Permission» и «Modify Permission» mark with a tick «extension/module/cs_info_blocks» );
  • In the list of modules «Extensions -> Extensions» install the module «Information Blocks».

Setting extension

To configure the module, go to «Extensions -> Extensions -> Information Blocks»

General settings:

  • Module name - name that will be displayed in the admin panel;
  • Status - module status is responsible for enabling settings;

Section settings:

  • Section title - title that will be displayed above the blocks;
  • Columns - which will be readable and the rest of the text will be hidden;
  • Image dimensions: - image width / height, or disable image resizing - 0x0;
  • Description length - number of characters displayed for description;
  • Description status - description display status;
  • Link to image / icon: - the link is set to an image or icon;
  • Link to title - link is set to heading;
  • Link as button - The link is set as a button.

Items:

  • Select Information: - choose from the ones created in the "Catalog -> Information" section;
  • Image / icon: - choose whether to show images, icon or disable;
  • Image - set image;
  • Class icon - icon classes can be viewed on the FontAwesome site;
  • Sort: - block order;
  • Information Title: - option to override the title of the article;
  • Information Description: - option to override article description.

Access / Design:

  • Access - to whom this block will be visible: for all users / for registered users only / for guests only;
  • Customer groups - manage access to customer groups;
  • Template - After creating a template for a module, you can select it here.

Template

Create a module template and place it in the folder:

« catalog / view / theme / default / template / custom_templates / module / info_blocks / file name in latin .(twig.tpl) »

As a basis, you can take the standard template on the path:

« catalog / view / theme / default / template / extension / module / cs_info_blocks.(twig.tpl) »

Stylization

If the default design doesn't fit into your website design, you can tweak the CSS styles in the «cs_info_blocks.min.css» file located at:

« catalog / view / cs_libs / css / cs_info_blocks.min.css »
