Единицы измерения товара

Позволяет добавлять и настраивать различные единицы измерения цен для товаров и опций. Подойдет, если вы продаете товар, цена на который устанавливается за определенное его количество, то есть цена за кв.м., пог.м, м3, кг, шт. и т.д.

Module capabilities

  • Localization of 3 languages: Ukrainian, English, Russian;
  • Мультиязычность;
  • Возможность указать в карточке товара единицу измерения;
  • Возможность указать единицу измерения для каждой опции товара;
  • Возможность указать одну единицу измерения по-умолчанию для всех товаров;
  • Изменение единицы измерения при выборе опции;
  • Единицы выводятся только для информативности на станице товара, категорий, поиска, производителях, модулях.

Extension installer

Important!

When the module is installed for the first time, additional fields are automatically installed in the database tables. Just in case, make a backup of the database.

The modification does not install?

Install «Localcopy OCMOD», then try to install the modification again.

Download

Modification installation consists of two parts:

  • «Unit_Of_Measurement_Product_x_OpenCart_x.ocmod.zip» - contains the main functionality of the module and modifier;
  • «Unit_Of_Measurement_Product_Default_OpenCart_x.ocmod.zip» - contains a modifier to connect the template.

Modification installation:

  • В Админ-панели перейдите «Меню -> Расширения -> Установка расширений» и установите два архива;
  • Получаем сообщение об успешной установке, переходим «Меню -> Расширение -> Модификаторы» и в правом верхнем углу нажимаем кнопку «Обновить»;
  • После этого перейдите в «Меню -> Система -> Группы пользователей» и для администратора установить права доступа (в блоках «Разрешить просмотр» и «Разрешить редактировать» отметить галочкой «extension/module/cs_unit»);
  • В списке модулей «Меню -> Расширение -> Модули» установите модуль «Единицы измерения товара».

Добавление единицы измерения

Для того, что бы добавить единицу измерения в OpenCart, необходимо выполнить следующие действия.

Описание действий:

  • Перейдите на страницу списка единиц измерения «Меню -> Система -> Локализация -> Единицы Измерения Товара»;
  • Нажмите кнопку «Добавить»;
  • В открывшемся окне необходимо заполнить форму;
  • Нажимаем кнопку «Сохранить»;

Настройка модуля

Для настройки модуля перейдите «Меню -> Расширения -> Модули -> Единицы Измерения Товара»

  • Статус - статус модуля отвечает за отображение модуля на сайте;
  • Разделитель - который будет отображаться между ценой и единицей измерения [Пример: 10.00 / шт. | 10.00 за м.кв ].;
  • Отображать в опциях - статус модуля отвечает за отображение модуля на сайте;
  • Отображать в списке товаров (Админ-панель) - статус модуля отвечает за отображение модуля на сайте.

Stylization

Если требуется стилизация элементов, то в CSS-файле который подключен к шаблону добавьте классы с нужными вам атрибутами:

  • .separate - стилизация разделителя;
  • .unit - стилизация ед. измерения;
  • .separate-option - стилизация разделителя в опциях;
  • .unit-option - стилизация ед. измерения в опциях.
