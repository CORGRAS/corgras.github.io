Extension capabilities

  • Localization of 3 languages: Ukrainian, English, Russian;
  • Management of the «Read More» script for all categories or individually;
  • Control on which screen resolutions the script will run.

Extension installer

Important!

When the module is installed for the first time, additional fields are automatically installed in the database tables. Just in case, make a backup of the database.

The modification does not install?

Install «Localcopy OCMOD», then try to install the modification again.

Download
  • In the admin panel go to «Menu -> Extensions -> Installer» and install the file «CD_Readmore_x_OpenCart_x.ocmod.zip»;
  • We get a message about successful installation, go to «Menu -> Extensions -> Modifications» and in the upper right corner click the button «Update»;
  • After go to «Menu -> System -> User Groups» and set permissions for administrator (in blocks «Access Permission» and «Modify Permission» mark with a tick «extension/module/cs_cd_readmore» );
  • In the list of modules «Menu -> Extensions -> Extensions» install the module «JS Readmore for category description».

Setting extension

To configure the extension, go to «Menu -> Extensions -> Extensions -> JS Readmore for category description».

Description settings:

  • Status - is responsible for enabling the extension;
  • «Read More» Button status - enable / disable script for all category descriptions;
  • Visible text height - which will be readable and the rest of the text will be hidden;
  • Screen resolution - at what screen resolutions the script will work.

Status DB

  • Display the status of fields in a database table;
  • Control setting or removing fields from the database.

Settings in categories

To customize the extension for a specific category, go to «Menu -> Catalog -> Categories» and select the desired category. Additional options are in the «Design» tab.

  • «Read More» Button status - enable / disable the script for a specific category or leave it by default, then the settings from the module are applied;
  • Visible text height - which will be readable and the other part of the text will be hidden.

Stylization

If the standard design of the «Expand / Collapse» button does not fit into your site design, you can tweak the CSS styles in the file «cs_cd_readmore.min.css», which is:

« catalog / view / javascript / cs_libs / css / cs_cd_readmore.min.css »
