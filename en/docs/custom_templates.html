Custom templates

Allows you to set a specific template for a product, category, manufacturer (brand) or information pages. You can also assign one template to all products under a certain category, and optionally override this template for a specific product

Version: 1.1
OpenCart: 3.x
Release: 5.12.2022

Warning

Note that when TEMPLATE is mentioned, these are the individual template files in your theme, NOT the entire theme. You cannot set different themes for each element with this extension.

If you installed a third-party theme on Opencart 3.x with file extension «.tpl»
The modifier will NOT work.

The generated files must have a file extension «.twig»

Extension installer

  • In the admin panel go to «Extensions -> Installer» and select the file: «Custom_Templates_v1.1_Opencart_3.ocmod.zip»;
  • We get a message about successful installation, go to «Extensions -> Modifications» and in the upper right corner click the button «Update»;
  • After go to «System -> User Groups» and set permissions for administrator
    (in blocks «Access Permission» и «Modify Permission» mark with a tick «extension/module/cs_custom_templates»);
  • In the list of modules «Extensions -> Extensions» install the module «Custom Templates».

Setting extension

After installation, the «custom_templates» folder should appear on the server at this path:
«catalog\view\theme\default\template\custom_templates\».

If you are using a different theme, you need to create this folder and additional folders in it «category», «information», «manufacturer», «product».

  • In a certain folder, place the file with the changes made (the file name must be in Latin and with the extension «.twig»);
  • In the admin panel, go to the section for which you created the template (products, categories, informational articles, manufacturers);
  • Switch to edit mode and in the «Design» tab, select a template from the list;
  • In a category, you can assign a template for all products that will be in this category;
  • In products, you can assign a new or reassign a template;
  • In manufacturers and informational articles, you can assign a new template.
