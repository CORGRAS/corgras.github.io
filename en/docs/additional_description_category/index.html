ADDITIONAL DESCRIPTION CATEGORY - Documentation

Created: 13/11/2017 Latest update: 10/03/2022 Version 2.0

This modifier will help you add text, banner, or HTML code at the bottom of category pages that will differ from the default category description. The modifier also has the ability to hide part of the long text for the upper and lower descriptions.

Warning

The modifier was tested on a standard template and on a clean system Opencart and ocStore.
Some sites may need some improvements.

Installation

Installing the modifier consists of two parts.

«AD_Category.ocmod.zip» - contains the main functionality.

«AD_Category_FrontEnd_Default.ocmod.zip» - modifier for displaying the description in the site template.

IMPORTANTLY!!!

When installing the module for the first time, additional fields are automatically installed in the database tables. Just in case, make a database backup

1. In the admin panel, go to «Extensions > Extension Installer» and install the file «AD_Category.ocmod.zip»;

2. We get a message about successful installation and go to «Extensions > Modifications» and click on the button «Refresh» in the upper-right corner [screenshot];

3. In the admin panel, go to «Extensions > Extension Installer» and install the file «AD_Category_FrontEnd_Default.ocmod.zip»;

4. We get a message about successful installation and go to «Extensions > Modifications» and click on the button «Refresh» in the upper-right corner [screenshot];

5. In the list of modules «Extensions > Extensions» install the module «Additional Description Category»

Module settings

To configure the module, go to:
«Extensions > Additional Description Category»

«General settings»:

Description on pagination pages - display text on the first page or on all;

Screen resolution - at what screen resolutions the script will run;

Animation speed - animation speed when expanding and collapsing a block;

Status - module status is responsible for enabling settings;

«Description settings»

Description positions - swap the top and bottom descriptions in places;

Top/bottom description status - display of the description block;

'Details' button status - enable/disable the script for all category descriptions;

Visible text height - which will be readable and the other part of the text will be hidden. For all category descriptions;

Image status - display the category image next to the description;

«Status DB»

Display the status of fields in a table. You can also manage to install / remove from the database..

Setting up in categories

If “Details button status” is disabled in the module settings, you can activate it inside the category in the “Design” tab:

Status of the 'Read more' button - enable/disable the script;

Height visible description - which will be readable and the rest of the text will be hidden;

Appearance

You can change the layout of blocks with a description in the files that are located:

« \ catalog \ view \ theme \ default \ template \ custom_templates \ addons \ category_desc_top.twig »

« \ catalog \ view \ theme \ default \ template \ custom_templates \ addons \ category_desc_bottom.twig »

Stylization

If the standard design of the "Expand/Collapse" button does not fit into your site design, you can tweak the CSS styles in the "readmore.css" file, which is located:

« catalog \ view \ javascript \ jquery \ readmore \ readmore.css »
