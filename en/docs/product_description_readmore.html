JS Readmore for product description

Allows you to hide part of a long text, with the ability to expand / collapse the product description

Module capabilities

  • Localization of 3 languages: Ukrainian, English, Russian;
  • Management of the «Read More» script for all products or individually;
  • Control on which screen resolutions the script will run.

Extension installer

Important!

When the module is installed for the first time, additional fields are automatically installed in the database tables. Just in case, make a backup of the database.

The modification does not install?

Install «Localcopy OCMOD», then try to install the modification again.

  • In the admin panel go to «Extensions -> Installer» and install the file «PD_Readmore_v1.2_OpenCart_x.ocmod.zip»;
  • We get a message about successful installation, go to «Extensions -> Modifications» and in the upper right corner click the button «Update»;
  • After go to «System -> User Groups» and set permissions for administrator (in blocks «Access Permission» и «Modify Permission» mark with a tick «extension/module/cs_pd_readmore» );
  • In the list of modules «Extensions -> Extensions» install the module «JS Readmore for product description».

Setting extension

To configure the module, go to «Extensions -> Extensions -> JS Readmore for product description»

Description settings:

  • Status - module status is responsible for enabling settings.
  • «Read More» Button status - enable / disable script for all product descriptions;
  • Visible text height - which will be readable and the rest of the text will be hidden;
  • Screen resolution - at what screen resolutions the script will work.

Status DB

  • Display the status of fields in a database table;
  • Control setting or removing fields from the database.

Settings in products

Additional settings are in the «Design» tab:

  • «Read More» Button status - enable / disable the script for a specific product or leave it by default, then the settings from the module are applied;
  • Visible text height - which will be readable and the other part of the text will be hidden.

Stylization

If the standard design of the «Expand / Collapse» button does not fit into your site design, you can tweak the CSS styles in the file «cs_pd_readmore.min.css», which is:

« catalog / view / javascript / cs_libs / css / cs_pd_readmore.min.css »
