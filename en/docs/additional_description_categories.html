Additional description of categories + settings

Allows you to add text, banner or HTML code at the bottom of the category pages, which will be different from the default description. Also, the modifier can hide part of the long text for the top and bottom descriptions and other settings

Version: 3.0
OpenCart: 2.1 | 2.2 | 2.3 | 3.x
Release: 30.11.2022

Module capabilities

  • Adds a title and description at the bottom of the page;
  • Localization of 3 languages: Ukrainian, English, Russian;
  • Manage top and bottom descriptions;
  • Manage category images;
  • Managing the «Read More» script for all categories or individually;
  • Replace descriptions;
  • Control on which screen resolutions the script will run;
  • Hide description on all pages except the first.

Extension installer

IMPORTANT!

When the module is installed for the first time, additional fields are automatically installed in the database tables. Just in case, make a backup of the database.

Modification installation consists of two parts:

  • «AD_Category_v3_OpenCart_x.ocmod.zip» - contains the main functionality of the module and modifier;
  • «AD_Category_FrontEnd_Default_OpenCart_x.ocmod.zip» - modifier to connect to the site template.

Modification installation:

  • In the admin panel go to «Extensions -> Installer» and install the files «AD_Category_v3_OpenCart_x.ocmod.zip» and «AD_Category_FrontEnd_Default_OpenCart_x.ocmod.zip»;
  • We get a message about successful installation, go to «Extensions -> Modifications» and in the upper right corner click the button «Update»;
  • After go to «System -> User Groups» and set permissions for administrator (in blocks «Access Permission» и «Modify Permission» mark with a tick «extension/module/cs_ad_category» );
  • In the list of modules «Extensions -> Extensions» install the module «Additional Description Categories».

Setting extension

To configure the module, go to «Extensions -> Extensions -> Additional Description Categories»

General settings:

  • Description positions - allows you to swap the top and bottom descriptions;
  • Status - module status is responsible for enabling settings.

Top and bottom description settings:

  • Description status - displaying a block with a description on the site;
  • Image status - display category image next to text;
  • Image position - which side of the text to display the image on;
  • Description pagination pages - show text on the first page or on all pages;
  • «Read More» Button status - enable / disable script for all category descriptions;
  • Visible text height - which will be readable and the rest of the text will be hidden;
  • Screen resolution - at what screen resolutions the script will work.

Status DB

  • Display the status of fields in a database table;
  • Control setting or removing fields from the database.

Setting in categories

Additional settings are in the «Design» tab

  • «Read More» Button status - enable / disable the script for a specific category or leave it by default, then the settings from the module are applied;
  • Visible text height - which will be readable and the other part of the text will be hidden.

Stylization

You can change the layout of blocks with a description in the files that are located.:

« / catalog / view / theme / default / template / custom_templates / addons / ad_category_top.(twig/tpl) »

« / catalog / view / theme / default / template / custom_templates / addons / ad_category_bottom.(twig/tpl) »

If the standard design of the «Expand / Collapse» button does not fit into your site design, you can tweak the CSS styles in the file «cs_ad_category.min.css», which is:

« catalog / view / cs_libs / css / cs_ad_category.min.css »
