Module capabilities

  • Localization of 3 languages: Ukrainian, English, Russian;
  • «Read More» Script Control;
  • Control on which screen resolutions the script will run;
  • H1-H6 heading level control;
  • Using a custom template for a module;
  • Control access to content;
  • Inserting images for different screen sizes;
  • Image size control New;
  • Delayed loading of images by means of the browser;
  • Insert video;
  • Position of the content in relation to the image or video;
  • Multilanguage.

Module installation

The module does not install?

Install «Localcopy OCMOD», then try to install the module again.

Download
  • In the admin panel go to «Menu -> Extensions -> Installer» and install the file «HTML_Content_Plus_x_OpenCart_x.ocmod.zip»;
  • We get a message about successful installation, go to «Menu -> System -> User Groups» and set permissions for administrator (in blocks «Access Permission» and «Modify Permission» mark with a tick «extension/module/cs_html_content_plus» );
  • In the list of modules «Menu -> Extensions -> Extensions» install the module «HTML-Content PLUS».

Module configuration

To configure the module, go to «Menu -> Extensions -> Extensions -> HTML-Content PLUS»:

General:

  • Module name - name that will be displayed in the admin panel;
  • Status - responsible for enabling the module;
  • Image - insert images for different screen sizes or one for all;
  • Video list - choose where the video will be inserted;
  • Video link - for supported services, it is enough to insert any video link, for others, copy links from « iframe src="" ».

Settings:

  • Content position - location of the text in relation to the image or video;
  • Header level - the heading level to display on the site;
  • «Read More» Button status - enable / disable the script;
  • Visible text height - which will be readable and the rest of the text will be hidden;
  • Screen resolution - at what screen resolutions the script will work;
  • Resize large image - enable custom resizing / disable to display original size;
  • Size large image (W x H) - for larger screens;
  • Resize medium image - enable custom resizing + if an image for medium screens is not added, it will be created from a large one. Turn off to display the original size;
  • Size medium image (W x H) - for medium screens;
  • Resize small image - enable custom resizing + if an image for small screens is not added, it will be created from a large one. Turn off to display the original size.;
  • Size small image (W x H) - for small screens.

Access:

  • Access - to whom this block will be visible: for all users / for registered users only / for guests only;
  • Customer groups - manage access to customer groups.

Design:

  • Template - After creating a template for a module, you can select it here.

Template

Create a module template and place it in the folder:

« catalog / view / theme / default / template / custom_templates / module / html_content_plus / file name in latin .(twig.tpl) »

As a basis, you can take the standard template on the path:

« catalog / view / theme / default / template / extension / module / cs_html_content_plus.(twig.tpl) »

Stylization

If the standard design of the «Expand / Collapse» button does not fit into your site design, you can tweak the CSS styles in the «cs_html_content_plus.min.css» file, which is located:

« catalog / view / javascript / cs_libs / css / cs_html_content_plus.min.css »
