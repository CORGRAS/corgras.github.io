Readmore.js

A smooth and responsive plugin written in Vanilla JS for collapsing and expanding long text blocks with «Read more» and «Close» links.

GitHub v.1.2
Documentation

«Readmore.js» - This is a plugin that allows hiding and displaying additional content on a web page, such as text, images, and other content, to improve readability and save space. It adds a «Read more» button that users can click to reveal hidden content. This is especially useful on mobile devices or for managing long texts on pages.

Expandable Sections

«Readmore.js» allows you to create expandable and collapsible content sections. You can display only a small excerpt of the text, and users can expand the full content as needed.

Customization

The plugin offers extensive options for customizing the appearance and behavior. You can define the height of the expanded and collapsed states, animation speed, and even replace the button text with your own.

Adaptability

«Readmore.js» is adaptive and can be activated only at a specific screen width. This allows you to effectively manage content display on various devices.

Accessibility

The full content is always accessible to search engines, contributing to the SEO optimization of your website.
