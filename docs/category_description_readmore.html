Встановлення розширення

Важливо!

При першому встановленні модуля автоматично встановлюються додаткові поля в таблиці БД.
Про всяк випадок робіть бекап БД.

Модифікатор не встановлюється?

Встановіть «Localcopy OCMOD», після чого спробуйте встановити модифікатор знову.

Завантажити
  • В Адмін-панелі перейдіть «Меню -> Розширення -> Встановлення розширень» та встановіть файл «CD_Readmore_x_OpenCart_x.ocmod.zip»;
  • Отримуємо повідомлення про успішне встановлення, переходимо «Меню -> Розширення -> Модифікатори» і в правому верхньому куті натискаємо кнопку «Оновити»;
  • Після цього перейдіть до «Меню -> Система -> Групи користувачів» і для адміністратора встановити права доступу (в блоках «Дозволити перегляд» та «Дозволити редагувати» відзначити галочкою «extension/module/cs_cd_readmore» );
  • У списку модулів «Меню -> Розширення -> Модулі» встановіть модуль «JS Readmore Для Опису Категорії».

Налаштування розширення

Для налаштування розширення, перейдіть «Меню -> Розширення -> Модулі -> JS Readmore Для Опису Категорії»

Опис налаштування:

  • Статус - відповідає за включення розширення;
  • Статус кнопки «Детальніше» - увімкнення / вимкнення скрипту для всіх описів категорій;
  • Висота видимого тексту - яка буде читана, а інша частина тексту буде прихована;
  • Розширення екрану - на яких дозволах екрану працюватиме скрипт.

Статус БД:

  • Відображення статусу полів у таблиці БД;
  • Керування установкою або видаленням полів з БД.

Налаштування у категоріях

Щоб налаштувати розширення для конкретної категорії, перейдіть «Меню -> Каталог -> Категорії», та виберіть потрібну категорію. Додаткові параметри знаходяться у вкладці «Дизайн»

  • Статус кнопки «Детальніше» - увімкнути / вимкнути скрипт для конкретної категорії або залишити за замовчуванням, тоді застосовуються налаштування з модуля;
  • Висота видимого тексту - яка буде читана, а інша частина тексту буде прихована.

Стилізація

Якщо стандартний дизайн кнопки «Розгорнути / Згорнути» не вписується у ваш дизайн сайту, ви можете підправити CSS стилі у файлі «cs_cd_readmore.min.css», який знаходиться:

« catalog / view / javascript / cs_libs / css / cs_cd_readmore.min.css »

Проблеми та рішення

Якщо виникли питання або проблеми, то ми зібрали в даному розділі відповіді на питання, що часто ставляться, вирішення найбільш загальних і часто зустрічаються проблем.
