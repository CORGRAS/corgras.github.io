О модуле

Данный модификатор позволит вам отвечать на отзывы клиентов. Также модификатор позволит вам:

  • Оставить ответ на отзыв;
  • Быстро изменять статус отзыва;
  • Отправить уведомление клиенту об ответе по почте;
  • Изменять дату добавления отзыва (для версий ниже 2.3.x);
  • Включать / отключать модерацию отзывов;
  • Включать / отключать обязательное заполнения e-mail;
  • WYSIWYG-редактор для редактирование отзывов.

Предупреждение

Модификатор тестировался на стандартном шаблоне и на чистой системе Opencart и ocStore.
Для некоторых сайтов могут понадобиться некоторые доработки.

Установка

Установка модификатора состоит из двух частей. В каждом архиве«Install Opencart» вы обнаружите модификаторы:

  • «ReplyToReviews_AdminPanel.ocmod.zip» - содержит основной функционал для Админ-панели.

    В архиве находится SQL файл. Если преффикс базы данных у Вас отличается от стандартного «oc_», то перед установкой извлеките файл «install.sql» и через тексторый редактор замените преффикс «oc_» на свой, после чего загрузите файл обратно в архив.

  • «ReplyToReviews_StoreFront_Default.ocmod.zip» - функционал для шаблона по умолчанию.
  • «ReplyToReviews_StoreFront_Journal.ocmod.zip» - функционал для шаблона Journal.

Шаг 1

  • В админке перейдите «Модули > Установка расширений» и установите файл «ReplyToReviews_AdminPanel.ocmod.zip»;
  • Получаем сообщение об успешной установке, переходим в «Модули > Модификаторы» и кликаем по кнопке «Обновить» в правом верхнем углу См. пример;
  • Если не появился функционал, проверяем лог на ошибки.

Шаг 2

Выберите модификатор для вашего шаблона и выполните следующие действия:

  • В админке перейдите «Модули > Установка расширений» и установите файл «ReplyToReviews_StoreFront_*.ocmod.zip» для вашего шаблона;
  • Получаем сообщение об успешной установке и идем в «Модули > Модификаторы» и кликаем по кнопке «Обновить» в правом верхнем углу См. пример;
  • Возникли проблемы / ошибки, проверяем лог на ошибки.

Настройки

Для настройки перейдите: «Настройки > Настройки магазина» внутри перейдите во вкладку «Опции», в разделе «Отзывы» найдете дополнительные настройки:

  • Отключить модерацию» - Отключает одобрение отзывов администратором;
  • Обязательное поле Email? - Включает или отключает обязательность заполнения поля Email;

Внешний вид

Если стандартный стили не вписываются в ваш дизайн сайта, вы можете подправить CSS стили в файле «reply_reviews.css», который находится:

catalog / view / stylesheet / reply_reviews.css

Локализация

Модификатор имеет полную поддержку нескольких языков. По умолчанию устанавливается английский, украинский и русский язык.
Ниже вы найдете инструкцию, как добавить и перевести модификатор на свой язык.

В зависимости от версии OpenCart папка может называться «russian» или «ru-ru»


Админ-панель:

  • Скопируйте файл «reply_reviews.php» из папки:

    admin / language / russian / catalog / reply_reviews.php

  • Вставьте в папку и после чего переведите содержимое файла:

    admin / language / ВАШ_ЯЗЫК / catalog / reply_reviews.php


Уведомление:

  • Скопируйте файл «reply_reviews.php» из папки:

    admin / language / russian / mail / reply_reviews.php

  • Вставьте в папку и после чего переведите содержимое файла:

    admin / language / ВАШ_ЯЗЫК / mail / reply_reviews.php


Витрина:

  • Скопируйте файл «reply_reviews.php» из папки:

    catalog / language / russian / product / reply_reviews.php

  • Вставьте в папку и после чего переведите содержимое файла:

    catalog / language / ВАШ_ЯЗЫК / product / reply_reviews.php
