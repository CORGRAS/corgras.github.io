Встановлення розширення

Модуль не встановлюється?

Встановіть «Localcopy OCMOD», після чого спробуйте встановити модуль знову.

  • В Адмін-панелі перейдіть «Розширення -> Встановлення розширень» та встановіть файл «HTML_Content_Plus_v5_OpenCart_x.ocmod.zip»;
  • Отримуємо повідомлення про успішне встановлення, переходимо «Система -> Групи користувачів» і для адміністратора встановити права доступу (в блоках «Дозволити перегляд» та «Дозволити редагувати» відзначити галочкою «extension/module/cs_html_content_plus» );
  • У списку модулів «Розширення -> Модулі» встановіть модуль «Текстовий блок HTML PLUS».

Налаштування розширення

Для налаштування модуля перейдіть «Розширення -> Модулі -> Текстовий блок HTML PLUS»

Загальні налаштування:

  • Назва модуля - назва, яка відображатиметься в адмін-панелі;
  • Статус - статус модуля відповідає за увімкнення налаштувань;
  • Зображення - вставляння зображень для різних розмірів екранів або одне для всіх.;
  • Відео список - вибір звідки буде вставлено відео;
  • Посилання на відео - для сервісів, що підтримуються, достатньо вставити будь-яке посилання на відео для інших скопіювати посилання з « iframe src="" »;
  • Позиція контенту - розташування тексту щодо зображення чи відео;
  • Статус кнопки «Детальніше» - увімкнення / вимкнення скрипту;
  • Висота видимого тексту - яка буде читана, а інша частина тексту буде прихована;
  • Розширення екрану - на яких дозволах екрану працюватиме скрипт;
  • Рівень заголовка - рівень заголовка для відображення на сайті;
  • Доступ - кому буде видно цей блок: для всіх користувачів / тільки для зареєстрованих / тільки для гостей;
  • Групи покупців - управління доступом до груп клієнтів;
  • Шаблон - після створення шаблону для модуля, можете вибрати його тут.

Шаблон

Створіть шаблон модуля і розмістіть його в папці:

« catalog / view / theme / default / template / custom_templates / module / html_content_plus / ім'я файлу на латиниці .(twig.tpl) »

За основу можете взяти стандартний шаблон на шляху:

« catalog / view / theme / default / template / extension / module / cs_html_content_plus.(twig.tpl) »

Стилізація

Якщо стандартний дизайн кнопки «Розгорнути / Згорнути» не вписується у ваш дизайн сайту, ви можете підправити CSS стилі у файлі «cs_html_content_plus.min.css», який знаходиться:

« catalog / view / javascript / cs_libs / css / cs_html_content_plus.min.css »
