Текстовий блок HTML PLUS

Дозволяє відображати довільний HTML-код на вашому веб-сайті. З можливістю керування видимістю тексту, приховуючи або розгортаючи його, і налаштовувати відображення для різних груп покупців

Можливості модуля

  • Локалізація 3 мов: Українська, Англійська, Російська;
  • Керування скриптом «Детальніше»;
  • Управління на які дозволи екранів буде працювати скрипт;
  • Управління рівнем заголовка H1-H6;
  • Використання кастомного шаблону для модуля;
  • Управлінням доступом до контенту;
  • Вставляння зображень для різних розмірів екранів;
  • Керування розмірами зображення Нове;
  • Відкладене завантаження зображень засобами браузера;
  • Вставлення відео;
  • Розташування контенту щодо зображення чи відео;
  • Мультимовність.

Встановлення модуля

Модуль не встановлюється?

Встановіть «Localcopy OCMOD», після чого спробуйте встановити модуль знову.

Завантажити
  • В Адмін-панелі перейдіть «Меню -> Розширення -> Встановлення розширень» та встановіть файл «HTML_Content_Plus_x_OpenCart_x.ocmod.zip»;
  • Отримуємо повідомлення про успішне встановлення, переходимо «Меню -> Система -> Групи користувачів» і для адміністратора встановити права доступу (в блоках «Дозволити перегляд» та «Дозволити редагувати» відзначити галочкою «extension/module/cs_html_content_plus» );
  • У списку модулів «Меню -> Розширення -> Модулі» встановіть модуль «Текстовий блок HTML PLUS».

Налаштування модуля

Для налаштування модуля перейдіть «Меню -> Розширення -> Модулі -> Текстовий блок HTML PLUS»

Загальні:

  • Назва модуля - назва, яка відображатиметься в адмін-панелі;
  • Статус - відповідає за увімкнення модуля;
  • Зображення - вставляння зображень для різних розмірів екранів або одне для всіх;
  • Відео список - вибір звідки буде вставлено відео;
  • Посилання на відео - для сервісів, що підтримуються, достатньо вставити будь-яке посилання на відео для інших скопіювати посилання з « iframe src="" ».

Налаштування:

  • Позиція контенту - розташування тексту щодо зображення чи відео;
  • Рівень заголовка - рівень заголовка для відображення на сайті;
  • Статус кнопки «Детальніше» - увімкнення / вимкнення скрипту;
  • Висота видимого тексту - яка буде читана, а інша частина тексту буде прихована;
  • Розширення екрану - на яких дозволах екрану працюватиме скрипт;
  • Змінити розмір великого зображення - увімкнення зміни розміру під власні / вимкнення для відображення вихідного розміру;
  • Розмір великого зображення (Ш x В) - для великих екранів;
  • Змінити розмір середнього зображення - увімкнення зміни розміру під власні + якщо зображення для середніх екранів не додано, то буде створено з великого. Вимкнення для відображення вихідного розміру;
  • Розмір середнього зображення (Ш x В) - для середніх екранів;
  • Змінити розмір маленького зображення - увімкнення зміни розміру під власні + якщо зображення для маленьких екранів не додано, то буде створено з великого. Вимкнення для відображення вихідного розміру;
  • Розмір маленького зображення (Ш x В) - для маленьких екранів.

Доступ:

  • Доступ - кому буде видно цей блок: для всіх користувачів / тільки для зареєстрованих / тільки для гостей;
  • Групи покупців - управління доступом до груп клієнтів.

Дизайн:

  • Шаблон - після створення шаблону для модуля, можете вибрати його тут.

Шаблон

Створіть шаблон модуля і розмістіть його в папці:

« catalog / view / theme / default / template / custom_templates / module / html_content_plus / ім'я файлу на латиниці .(twig.tpl) »

За основу можете взяти стандартний шаблон на шляху:

« catalog / view / theme / default / template / extension / module / cs_html_content_plus.(twig.tpl) »

Стилізація

Якщо стандартний дизайн кнопки «Розгорнути / Згорнути» не вписується у ваш дизайн сайту, ви можете підправити CSS стилі у файлі «cs_html_content_plus.min.css», який знаходиться:

« catalog / view / javascript / cs_libs / css / cs_html_content_plus.min.css »

Проблеми та рішення

Якщо виникли питання або проблеми, то ми зібрали в даному розділі відповіді на питання, що часто ставляться, вирішення найбільш загальних і часто зустрічаються проблем.
