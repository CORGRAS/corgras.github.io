Встановлення розширення

ВАЖЛИВО!

При першому встановленні модуля автоматично встановлюються додаткові поля в таблиці БД. Про всяк випадок робіть бекап БД.

В Адмін-панелі перейдіть «Розширення -> Встановлення розширень»:

  • Встановіть «PD_Readmore_v1_OpenCart_x.ocmod.zip»;
  • Отримуємо повідомлення про успішне встановлення, переходимо «Розширення -> Модифікатори» і в правому верхньому куті натискаємо кнопку «Оновити»;
  • Після цього перейдіть до «Система -> Групи користувачів» і для адміністратора встановити права доступу (в блоках «Дозволити перегляд» та «Дозволити редагувати» відзначити галочкою «extension/module/cs_pd_readmore»);
  • У списку модулів «Розширення -> Модулі» встановіть модуль «JS Readmore Для Опису Товару».

Налаштування розширення

Для налаштування модуля перейдіть: «Розширення -> Модулі -> JS Readmore Для Опису Товару»

Опис налаштування:

  • Статус - статус модуля відповідає за увімкнення налаштувань.
  • Статус кнопки «Докладніше» - увімкнення / вимкнення скрипту для всіх описів товарів;
  • Висота видимого тексту - яка буде читана, а інша частина тексту буде прихована;
  • Розширення екрану - на яких дозволах екрану працюватиме скрипт.

Статус БД:

  • Відображення статусу полів у таблиці БД;
  • Керування установкою або видаленням полів з БД.

Налаштування у товарах

Додаткові параметри знаходяться у вкладці «Дизайн»:

  • Статус кнопки «Докладніше» - увімкнути / вимкнути скрипт для конкретного товару або залишити за замовчуванням, тоді застосовуються налаштування з модуля;
  • Висота видимого тексту - яка буде читана, а інша частина тексту буде прихована.

Стилізація

Якщо стандартний дизайн кнопки «Розгорнути / Згорнути» не вписується у ваш дизайн сайту, ви можете підправити CSS стилі у файлі «cs_pd_readmore.min.css», який знаходиться:

« catalog \ view \ cs_libs \ css \ cs_pd_readmore.min.css »
