Додатковий опис категоріям + налаштування

Дозволяє додавати текст, банер або HTML-код у нижній частині сторінок категорій, який відрізнятиметься від опису за промовчанням. Також модифікатор може приховати частину довгого тексту для верхнього та нижнього опису та інші налаштування

Можливості модуля

  • Додає заголовок та опис внизу сторінки;
  • Підтримка 3 мов: Українська, Англійська, Російська;
  • Керувати верхнім та нижнім описом;
  • Керування зображенням категорій;
  • Керування скриптом «Детальніше» для всіх категорій або індивідуально;
  • Міняти опис місцями;
  • Управління на які дозволи екранів буде працювати скрипт, для всіх або індивідуально;
  • Приховати опис на всіх сторінках, крім першої.

Для деяких сайтів можуть знадобитися деякі доопрацювання або адаптація під шаблон.

Встановлення розширення

ВАЖЛИВО!

При першому встановленні модуля автоматично встановлюються додаткові поля в таблиці БД. Про всяк випадок робіть бекап БД.

Встановлення модуля складається із двох частин:

  • «AD_Category_v3_OpenCart_x.ocmod.zip» - міститься основний функціонал модуля та модифікатора;
  • «AD_Category_FrontEnd_Default_OpenCart_x.ocmod.zip» - модифікатор для підключення до шаблону веб-сайту.

В Адмін-панелі перейдіть «Розширення -> Встановлення розширень»:

  • Встановіть «AD_Category_v3_OpenCart_x.ocmod.zip» та «AD_Category_FrontEnd_Default_OpenCart_x.ocmod.zip»;
  • Отримуємо повідомлення про успішне встановлення, переходимо «Розширення -> Модифікатори» і в правому верхньому куті натискаємо кнопку «Оновити»;
  • Після цього перейдіть до «Система -> Групи користувачів» і для адміністратора встановити права доступу (в блоках «Дозволити перегляд» та «Дозволити редагувати» відзначити галочкою «extension/module/cs_ad_category»);
  • У списку модулів «Розширення -> Модулі» встановіть модуль «Дополнительное Описание Категориям».

Налаштування розширення

Для налаштування модуля перейдіть: «Розширення -> Модулі -> Додаткове Опис Категоріям»

Загальні налаштування:

  • Позиції описів - дозволяє поміняти місцями верхній та нижній опис;
  • Статус - статус модуля відповідає за увімкнення налаштувань.

Налаштування верхнього та нижнього опису:

  • Статус опису - відображення блоку з описом на сайті;
  • Статус зображення - відображення зображення категорії біля тексту;
  • Позиція зображення - з якого боку тексту відображати зображення;
  • Опис на сторінках пагінації - відображення тексту на першій сторінці або на всіх;
  • Статус кнопки «Докладніше» - увімкнення/вимкнення скрипту для всіх описів категорій;
  • Висота видимого тексту - яка буде читана, а інша частина тексту буде прихована;
  • Розширення екрану - на яких дозволах екрану працюватиме скрипт.

Статус БД:

  • Відображення статусу полів у таблиці БД;
  • Керування установкою або видаленням полів з БД.

Налаштування у категоріях

Додаткові параметри знаходяться у вкладці «Дизайн»:

  • Статус кнопки «Докладніше» - увімкнути/вимкнути скрипт для конкретної категорії або залишити за замовчуванням, тоді застосовуються налаштування з модуля;
  • Висота видимого тексту - яка буде читана, а інша частина тексту буде прихована.

Стилізація

Змінити верстку блоків з описом можна у файлах, які знаходяться:

« \ catalog \ view \ theme \ default \ template \ custom_templates \ addons \ ad_category_top.(twig/tpl) »

« \ catalog \ view \ theme \ default \ template \ custom_templates \ addons \ ad_category_bottom.(twig/tpl) »

Якщо стандартний дизайн кнопки «Розгорнути / Згорнути» не вписується у ваш дизайн сайту, ви можете підправити CSS стилі у файлі «cs_ad_category.min.css», який знаходиться:

« catalog \ view \ cs_libs \ css \ cs_ad_category.min.css »
