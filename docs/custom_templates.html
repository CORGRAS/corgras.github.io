Можливості розширення

  • Призначити певний шаблон для товару, категорії, виробника (бренду) чи інформаційних сторінок;
  • Призначити один шаблон для всіх товарів під певною категорією;
  • Перевизначити шаблон для конкретного товару.

Встановлення розширення

Модифікатор не встановлюється?

Встановіть «Localcopy OCMOD», після чого спробуйте встановити модифікатор знову.

Завантажити
  • В Адмін-панелі перейдіть «Меню -> Розширення -> Встановлення розширень» та встановіть файл «Custom_Templates_x_Opencart_x.ocmod.zip»;
  • Отримуємо повідомлення про успішне встановлення, переходимо «Меню -> Розширення -> Модифікатори» і в правому верхньому куті натискаємо кнопку «Оновити»;
  • Після цього перейдіть до «Меню -> Система -> Групи користувачів» і для адміністратора встановити права доступу (в блоках «Дозволити перегляд» та «Дозволити редагувати» відзначити галочкою «extension/module/cs_custom_templates» );
  • У списку модулів «Меню -> Розширення -> Модулі» встановіть модуль «Користувальницькі шаблони».

Налаштування розширення

Після встановлення на сервері має з'явитися папка «custom_templates» по даному шляху: «catalog/view/theme/default/template/custom_templates/»

Якщо Ви використовуєте іншу тему, Вам необхідно створити цю папку і в ній додаткові папки «category», «information», «manufacturer», «product».

  • У певну папку розмістіть файл із внесеними змінами (назва файлу має бути на латиниці та з розширенням «.twig»);
  • В адмін-панелі перейдіть до розділу, для якого ви створили шаблон (товари, категорії, інформаційні статті, виробники);
  • Перейдіть до режиму редагування та у вкладці «Дизайн», виберіть зі списку шаблон;
  • У категорії можете призначити шаблон для всіх товарів, які будуть у цій категорії;
  • У товарах можете призначити новий або перепризначити шаблон;
  • У виробниках та інформаційних статтях можете призначити новий шаблон.

Якщо виникли питання або проблеми, то ми зібрали в даному розділі відповіді на питання, що часто ставляться, вирішення найбільш загальних і часто зустрічаються проблем.
