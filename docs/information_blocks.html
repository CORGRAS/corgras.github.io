Можливості модуля

  • Локалізація 3 мов: Українська, Англійська, Російська;
  • Призначити зображення або іконку для блоку;
  • Призначити кількість колонок для різної ширини екрану;
  • Перевизначити заголовок;
  • Перевизначити короткий опис;
  • Керувати кількістю символів для короткого опису;
  • Керувати розміром зображення;
  • Керувати посиланнями (зображення / іконка, заголовок, кнопка);
  • Керувати доступом та групами покупців;
  • Підключати шаблони до модуля;
  • Мультимовність.

Встановлення розширення

Модуль не встановлюється?

Встановіть «Localcopy OCMOD», після чого спробуйте встановити модуль знову.

  • В Адмін-панелі перейдіть «Розширення -> Встановлення розширень» та встановіть файл «Information_Blocks_v3_OpenCart_x.ocmod.zip»;
  • Отримуємо повідомлення про успішне встановлення, переходимо «Система -> Групи користувачів» і для адміністратора встановити права доступу (в блоках «Дозволити перегляд» та «Дозволити редагувати» відзначити галочкою «extension/module/cs_info_blocks» );
  • У списку модулів «Розширення -> Модулі» встановіть модуль «Інформаційні блоки».

Налаштування розширення

Для налаштування модуля перейдіть «Розширення -> Модулі -> Інформаційні блоки»

Загальні налаштування:

  • Назва модуля - назва, яка відображатиметься в адмін-панелі;
  • Статус - статус модуля відповідає за увімкнення налаштувань.

Налаштування розділу:

  • Назва розділу - заголовок, який будуть відображатися над блоками;
  • Колонки - яка буде читана, а інша частина тексту буде прихована;
  • Розміри зображення - ширина / висота зображення, або вимикає зміну розміру зображення - 0x0;
  • Кількість символів в описі - кількість знаків, що відображаються для опису;
  • Статус опису - статус відображення опису;
  • Посилання у вигляді зображення - посилання встановлюється на зображення або іконку;
  • Посилання у вигляді заголовка - посилання встановлюється на заголовок;
  • Посилання у вигляді кнопки - посилання встановлюється у вигляді кнопки.

Блоки:

  • Оберіть статтю - вибір зі створених у розділі «Категорії -> Статті»;
  • Зображення / іконка - вибір, показувати зображення, іконку або вимкнути;
  • Зображення - встановлення зображення;
  • Class іконки - класи іконок можна переглянути на сайті FontAwesome;
  • Сортування - порядок блоків;
  • Назва статті - можливість перевизначити заголовок статті;
  • Опис - можливість перевизначити опис статті.

Доступ / Дизайн:

  • Доступ - кому буде видно цей блок: для всіх користувачів / тільки для зареєстрованих / тільки для гостей;
  • Групи покупців - управління доступом до груп клієнтів;
  • Шаблон - після створення шаблону для модуля, можете вибрати його тут.

Шаблон

Створіть шаблон модуля і розмістіть його в папці:

« catalog / view / theme / default / template / custom_templates / module / info_blocks / ім'я файлу на латиниці .(twig.tpl) »

За основу можете взяти стандартний шаблон на шляху:

« catalog / view / theme / default / template / extension / module / cs_info_blocks.(twig.tpl) »

Стилізація

Якщо стандартний дизайн не вписується у ваш дизайн сайту, ви можете підправити стилі CSS у файлі «cs_info_blocks.min.css», який знаходиться:

« catalog / view / cs_libs / css / cs_info_blocks.min.css »
