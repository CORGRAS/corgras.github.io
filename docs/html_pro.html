Введение

Html Pro даёт возможность пользователю самому создавать сетку с неограниченным количеством текстовых блоков. Модуль не привязан к определенному CSS-фреймворку, тем самым дает возможность использовать любой CSS-фреймворк, CSS-сетку или применить свои CSS-стили для оформления, всего лишь назначив для блока нужный CLASS и/или ID.

  • Отображение на страницах сайта текстовые блоки;
  • Создание неограниченного количества блоков;
  • Использование иконочных шрифтов;
  • Не привязан к CSS-фреймворку (Bootstrap 3);
  • Мультиязычность;
  • Назначать: изображение и/или иконку;
  • Назначать class и/или id.

Установка

  1. Из папки «upload» скопируйте папки «admin» и «catalog»;
  2. В админ-панели перейти в «Система > Группы пользователей» и для администратора установить права доступа (в блоках «Разрешить просмотр» и «Разрешить редактировать» отметить галочкой extension/module/html_pro);
  3. В списке модулей «Расширения -> Модули» установите модуль HTML Pro;
  4. В «Дизайн -> Схема» выберите страницу магазина и расположите модуль в нужной позиции.

Настройка модуля

  • Название модуля - название которое будет отображаться в админ-панели
  • Статус - включение/выключение модуля

Общие настройки

  • Заголовок - текст над блоками
  • Class / ID для родительского DIV-а - родительский блок, который включает все блоки.
  • Class / ID для контейнера блоков - контейнер который будет включать в себя колонки

Блоки

  • Изображение- назначение изображения;
  • Class иконки- классы иконок можно посмотреть на сайте FontAwesome;
  • Статус - показывать или не показывать изображение или иконку
  • Заголовок - заголовок блока
  • Ссылка - ссылка в заголовке
  • Описание - текст блока
  • Сортировка - назначить порядок блоков

Настройки блоков Class / ID

  • Class / ID Родительский блок - для стилизации колонок
  • Class / ID Дочерний блок - дополнительный вложенный DIV
  • Class / ID Блок для изображения или иконки
  • Class / ID Родительский блок для заголовка и описания
  • Class / ID Блок для заголовока
  • Class / ID Блок для описания

Внешний вид

Модуль идет без стилизации блоков.

Для оформления блоков, Вам нужно взять готовые CSS стили или написать свои CSS стили и разместить в файле «stylesheet.css», который находится:

catalog \ view \ theme \ ВАША ТЕМА \ stylesheet \ stylesheet.css

Потом в настройках модуля, в нужных полях, назначить Class или Id.

Локализация

Модификатор имеет полную поддержку нескольких языков. По умолчанию устанавливается английский и русский язык.
Ниже вы найдете инструкцию, как добавить и перевести модификатор на свой язык.

В зависимости от версии OpenCart папка может называться «russian» или «ru-ru»


Админ-панель:

  • Скопируйте файл «html_pro.php» из папки:

    admin \ language \ russian \ module \ html_pro.php

  • Вставьте в папку и после чего переведите содержимое файла:

    admin \ language \ ВАШ_ЯЗЫК \ module \ html_pro.php
